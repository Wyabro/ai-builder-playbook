# Docs

The rendered guide lives in [`index.md`](./index.md).

The current version of the guide is v4, with a dedicated page at [`v4/index.md`](./v4/index.md). When GitHub Pages is enabled, that page is served at `https://wyabro.github.io/ai-builder-playbook/v4/` so it can be linked or bookmarked independently of the homepage.

GitHub can render the main guide directly from the repository `README.md`. The `docs/` copy exists so the repository can also be used with GitHub Pages or other static documentation tools.
