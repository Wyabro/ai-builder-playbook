# Contributing

This repository contains a practical guide to making games and apps with AI.

Suggested contribution style:

- Keep changes practical and builder-focused.
- Prefer clear explanations over compressed lists.
- Add sources for current factual claims.
- Avoid tool hype unless the tool changes the workflow in a durable way.
- Preserve the guide’s chronological flow from idea to shipped product.

For larger revisions, open an issue first with:

1. The section you want to improve.
2. The problem you see.
3. The change you recommend.
4. Any sources or examples that support the change.
